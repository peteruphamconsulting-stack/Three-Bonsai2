# Three-Bonsai2
Reproduction code and certified data for the paper "A Geometric Sieve of Eratosthenes: Least-prime-factor densities, perfect packings, and the integers as a point set"

## Sharing files for build help

If you want help building this repository, add the files to the repository and share their paths.

1. Put each file in the repository with its intended name and folder structure.
2. Tell the assistant which files were added and what build command or entry point you expect to use.
3. If any file is generated, binary, or unusually large, include a short description of what it is for.
4. Do not share secrets, credentials, or private keys in the repository.

Example:

- Add source files under their expected folders.
- Say: "Please build the repository using the files at `/home/runner/work/Three-Bonsai2/Three-Bonsai2/<path>`."

If the files are not yet in the repository, open a pull request or otherwise add them to the working tree first so they can be inspected during the build.
